# <a name="database-providers"></a><span data-ttu-id="5b3ba-102">Поставщики баз данных</span><span class="sxs-lookup"><span data-stu-id="5b3ba-102">Database Providers</span></span>

<span data-ttu-id="5b3ba-103">Entity Framework Core поддерживает доступ к множеству разных баз данных с использованием библиотек подключаемых модулей, которые называются поставщиками баз данных.</span><span class="sxs-lookup"><span data-stu-id="5b3ba-103">Entity Framework Core can access many different databases through plug-in libraries called database providers.</span></span>

## <a name="current-providers"></a><span data-ttu-id="5b3ba-104">Текущие поставщики</span><span class="sxs-lookup"><span data-stu-id="5b3ba-104">Current providers</span></span>

> [!IMPORTANT]  
> <span data-ttu-id="5b3ba-105">Поставщики EF Core поступают из самых разных источников.</span><span class="sxs-lookup"><span data-stu-id="5b3ba-105">EF Core providers are built by a variety of sources.</span></span> <span data-ttu-id="5b3ba-106">Не все поставщики разрабатываются в рамках [проекта Entity Framework Core](https://github.com/aspnet/EntityFrameworkCore).</span><span class="sxs-lookup"><span data-stu-id="5b3ba-106">Not all providers are maintained as part of the [Entity Framework Core Project](https://github.com/aspnet/EntityFrameworkCore).</span></span> <span data-ttu-id="5b3ba-107">Выбирая поставщика, обязательно оцените качество, лицензирование, поддержку и другие показатели на соответствие вашим требованиям.</span><span class="sxs-lookup"><span data-stu-id="5b3ba-107">When considering a provider, be sure to evaluate quality, licensing, support, etc. to ensure they meet your requirements.</span></span> <span data-ttu-id="5b3ba-108">Также обязательно ознакомьтесь подробными сведениями о совместимости версий, представленными в документации по каждому поставщику.</span><span class="sxs-lookup"><span data-stu-id="5b3ba-108">Also make sure you review each provider's documentation for detailed version compatibility information.</span></span>

> [!IMPORTANT]
