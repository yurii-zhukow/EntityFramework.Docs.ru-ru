# <a name="database-providers"></a><span data-ttu-id="54249-102">Поставщики баз данных</span><span class="sxs-lookup"><span data-stu-id="54249-102">Database Providers</span></span>

<span data-ttu-id="54249-103">Entity Framework Core поддерживает доступ к множеству разных баз данных с использованием библиотек подключаемых модулей, которые называются поставщиками баз данных.</span><span class="sxs-lookup"><span data-stu-id="54249-103">Entity Framework Core can access many different databases through plug-in libraries called database providers.</span></span>

## <a name="current-providers"></a><span data-ttu-id="54249-104">Текущие поставщики</span><span class="sxs-lookup"><span data-stu-id="54249-104">Current providers</span></span>

> [!IMPORTANT]  
> <span data-ttu-id="54249-105">Поставщики EF Core поступают из самых разных источников.</span><span class="sxs-lookup"><span data-stu-id="54249-105">EF Core providers are built by a variety of sources.</span></span> <span data-ttu-id="54249-106">Не все поставщики разрабатываются в рамках [проекта Entity Framework Core](https://github.com/aspnet/EntityFrameworkCore).</span><span class="sxs-lookup"><span data-stu-id="54249-106">Not all providers are maintained as part of the [Entity Framework Core Project](https://github.com/aspnet/EntityFrameworkCore).</span></span> <span data-ttu-id="54249-107">Выбирая поставщика, обязательно оцените качество, лицензирование, поддержку и другие показатели на соответствие вашим требованиям.</span><span class="sxs-lookup"><span data-stu-id="54249-107">When considering a provider, be sure to evaluate quality, licensing, support, etc. to ensure they meet your requirements.</span></span> <span data-ttu-id="54249-108">Также обязательно ознакомьтесь подробными сведениями о совместимости версий, представленными в документации по каждому поставщику.</span><span class="sxs-lookup"><span data-stu-id="54249-108">Also make sure you review each provider's documentation for detailed version compatibility information.</span></span>

> [!IMPORTANT]  
> <span data-ttu-id="54249-109">Поставщики EF Core обычно совместимы с дополнительными номерами версий, а не с основными.</span><span class="sxs-lookup"><span data-stu-id="54249-109">EF Core providers typically work across minor versions, but not across major versions.</span></span> <span data-ttu-id="54249-110">Например, поставщик, выпущенный для EF Core 2.1, должен работать с EF Core 2.2, но не будет работать с EF Core 3.0.</span><span class="sxs-lookup"><span data-stu-id="54249-110">For example, a provider released for EF Core 2.1 should work with EF Core 2.2, but will not work with EF Core 3.0.</span></span>
