<span data-ttu-id="814cc-105">Entity Framework Core ([EF Core](xref:core/index)) — это современный модуль сопоставления "объект — база данных" для .NET.</span><span class="sxs-lookup"><span data-stu-id="814cc-105">Entity Framework Core ([EF Core](xref:core/index)) is a modern object-database mapper for .NET.</span></span> <span data-ttu-id="814cc-106">Он поддерживает запросы LINQ, отслеживание изменений, обновления и миграции схемы.</span><span class="sxs-lookup"><span data-stu-id="814cc-106">It supports LINQ queries, change tracking, updates, and schema migrations.</span></span>
