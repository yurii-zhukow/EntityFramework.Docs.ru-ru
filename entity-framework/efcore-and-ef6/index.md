## <a name="ef-core"></a><span data-ttu-id="bd529-104">EF Core</span><span class="sxs-lookup"><span data-stu-id="bd529-104">EF Core</span></span>

<span data-ttu-id="bd529-105">Entity Framework Core ([EF Core](xref:core/index)) — это современный модуль сопоставления "объект — база данных" для .NET.</span><span class="sxs-lookup"><span data-stu-id="bd529-105">Entity Framework Core ([EF Core](xref:core/index)) is a modern object-database mapper for .NET.</span></span> <span data-ttu-id="bd529-106">Он поддерживает запросы LINQ, отслеживание изменений, обновления и миграции схемы.</span><span class="sxs-lookup"><span data-stu-id="bd529-106">It supports LINQ queries, change tracking, updates, and schema migrations.</span></span>
