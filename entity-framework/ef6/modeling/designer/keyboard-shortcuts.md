# <a name="entity-framework-designer-keyboard-shortcuts"></a>Entity Framework Designer сочетаний клавиш
На этой странице представлен список сочетаний клавиш шоркутс, доступных на различных экранах Entity Framework Tools для Visual Studio.

## <a name="adonet-entity-data-model-wizard"></a>Мастер EDM ADO.NET

### <a name="step-one-choose-model-contents"></a>Шаг 1. Выбор содержимого модели

![Мастер 1](~/ef6/media/wizardone.png)

| Клавиша  | Действие                                                     | Примечания                                               |
|:----------|:-----------------------------------------------------------|:----------------------------------------------------|
| **Alt + n** | Перейти к следующему экрану                                        | Недоступно для всех элементов, выделенных для содержимого модели. |
| **Alt + f** | Завершите работу с мастером.                                              | Недоступно для всех элементов, выделенных для содержимого модели. |
| **Alt + w** | Переключить фокус на «что должна содержать модель?» . |                                                     |
